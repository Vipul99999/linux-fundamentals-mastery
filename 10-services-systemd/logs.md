# The Biggest Idea

Logs are not text files.

Logs are:

> Historical records that describe everything an operating system experiences.

Think:

```text
Logs = Memory
```

Without logs:

```text
Operating system

↓

Amnesia
```

# The Logging Pipeline

Every logging system has 5 components.

```text
Producer

Collector

Storage

Query

Analysis
```

---

# 1 Producer

Something creates an event.

Examples:

```text
Kernel

Nginx

Docker

SSH

Redis

PostgreSQL
```

---

# 2 Collector

Something gathers logs.

Examples:

```text

rsyslog
```

---

# 3 Storage

Something stores logs.

Examples:

```text
/var/log

/var/log/journal
```

---

# 4 Query

Something searches logs.

Examples:

```text

less
```

---

# 5 Analysis

Engineers investigate.

Questions:

```text

Patterns

Failures

Trends
```

---

# journald

journald is a daemon.

Responsibilities:

```text
Collect logs

Tag logs

Store logs

Index logs

Expose logs
```

---

# journalctl

journalctl is NOT a logger.

It is a reader.

Responsibilities:

```text
Query logs

Filter logs

Analyze logs
```

---

# rsyslog

Traditional log router.

Responsibilities:

```text
Receive logs

Forward logs

Write files

Send logs remotely
```

---
